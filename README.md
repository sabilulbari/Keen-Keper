# 📌 Relationship Tracker Dashboard

A modern, smart relationship management dashboard that helps users track interactions with friends, colleagues, and contacts. It visualizes communication history (call, text, video), highlights relationship status, and ensures you never lose touch with important people in your network.

---

## ⚙️ Tech Stack

* Next.js (App Router)
* React.js
* Tailwind CSS
* React Icons
* Context API

---

## 🚀 Features

### 📊 1. Interaction Timeline Tracking

Track and filter communication history by **Call, Text, and Video** in a clean timeline UI.

### 👤 2. Smart Relationship Dashboard

Each contact has a detailed profile including:

* Days since last contact
* Relationship status
* Tags
* Bio
* Communication goal tracking

### 📈 3. Insightful Analytics View

Displays structured relationship insights such as:

* Contact frequency
* Next due date
* Goal tracking
